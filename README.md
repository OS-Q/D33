### [节点描述](https://github.com/OS-Q/M1/wiki) 

数据转换节点，通过集成各种数据采集终端传感器，通过归一的传感器实现数据输出。

纯数据转换不带有业务控制逻辑，用于转换和采集数据，无策略无条件进行数据转换。
